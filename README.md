# express-demo-ui

## Use these commands to start this app on your machine

```cmd
$ git clone https://github.com/Yash-s-Code-Camp/express-demo-ui.git
$ cd express-demo-ui
$ npm install
```

> If you have already cloned it then use this command to update

```cmd
$ git pull origin
```

> To run the server

```cmd
$ npm start
```
